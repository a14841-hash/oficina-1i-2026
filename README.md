# 🐍 Exercícios de Condicionais em Python

Este repositório contém um conjunto de exercícios introdutórios em **Python**, focados no uso de **estruturas condicionais (`if`, `elif`, `else`)**. Estes exemplos fazem parte do meu percurso no curso **Técnico de Gestão e Programação de Sistemas Informáticos (TGPSI)**, onde estou a desenvolver competências fundamentais de programação e lógica computacional.

---

## 📌 Funcionalidades do Script

O código incluído neste repositório realiza várias operações básicas, entre elas:

- Verificar se um número é **positivo** ou **negativo**
- Determinar se uma pessoa é **maior** ou **menor de idade**
- Identificar se um número é **par** ou **ímpar**
- Comparar dois números e indicar o **maior**
- Validar uma **password**
- Avaliar uma **nota escolar** e classificá-la
- Classificar uma pessoa por **faixa etária**
- Verificar múltiplos de **3** e **5**
- Validar credenciais de utilizador
- Verificar se um número está entre **10 e 20**
- Encontrar o **maior de quatro números**
- Calcular o **IMC** e indicar a categoria correspondente
- Aplicar **descontos** com base no valor da compra
- Determinar se um ano é **bissexto**

---

São exercícios fundamentais para consolidar a base da programação e preparar para projetos mais avançados.
